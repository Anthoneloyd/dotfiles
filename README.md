# Bindings
- `␣pv` - :Ex - close editor.

## Telescope
- `␣pf` - project find.
- `␣ps` - project search (w grep).

## Harpoon
- `␣a` - add file.
- `ctrl + e` - quick menu.
- `ctrl + t` - next file.
- `ctrl + h` - previous file.
